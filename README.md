# LangGraph AI News to LinkedIn Pipeline

This project demonstrates a LangGraph pipeline that:

1. Retrieves the latest AI news from the Perplexity API.
2. Selects the best 5 stories.
3. Uses ChatGPT (via the OpenAI API) to generate an engaging LinkedIn post.
4. Posts the content to LinkedIn via its API.
